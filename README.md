# AstraOS Landing Page

Premium animated SaaS landing page built with Next.js, React, TypeScript, TailwindCSS, Framer Motion, GSAP, and Lucide Icons.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production checks

```bash
npm run typecheck
npm run build
```

## Structure

- `app/` contains the Next.js shell, metadata, and global styling.
- `components/landing-page.tsx` contains the reusable sections, animations, particle field, magnetic buttons, and product showcase.
- `public/og-image.svg` is the social preview asset.
